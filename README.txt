This repository includes code for Shelby McCahon's structural equation modeling analysis to better understand the direct and indirect effects of agriculture and drought on shorebird body condition and prey availability in the Prairie Pothole Region.

Date Created: August 12th, 2025

R scripts and descriptions: SEM_Data Preparation_2025-08-12 (data cleaning file)
                            01_SEM_Modeling_Building_2025-08-22 (first attempt at modeling but was running into model convergence issues because I'm using different datasets)
                            02_SEM_Model_Building_Complete_Dataset_2025-08-29 (I limited the dataset to subset that has information from all predictors)
